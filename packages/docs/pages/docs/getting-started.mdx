# Getting Started

### Getting the current user's public information

URL: `https://minet.mokshitjain.co/api/auth/me`

```ts
type User = {
  name: string;
  publicAddress: string;
  activity: Activity[];
  alerts: Alert[];
}
```

Example:

```js
await fetch('https://minet.mokshitjain.co/api/auth/me').then(r => r.json())
```

### Getting the user's activity stats

URL: `https://minet.mokshitjain.co/api/stats?user=[publicAddress]`

```ts
type Response = User & {
  activities: Activity[]
}
```

Example:

```js
await fetch('https://minet.mokshitjain.co/api/stats?user=[publicAddress]').then(r => r.json())
```

The `publicAddress` here refers to the wallet ID, which you can get from Metamask or similar extensions in the following way:

```js
const hash = await window.ethereum.request({ method: 'eth_requestAccounts' })

await fetch(`https://minet.mokshitjain.co/api/stats?user=${hash[0]}`).then(r => r.json())
```

### Get friendly messages using GPT3 based on activity

URL: `https://minet.mokshitjain.co/api/format-message`

```ts
type Request = {
  publicAddress: string;
  activities: Activity[];
}
type Response = {
  message: string;
}
```

Example:

```js
await fetch('https://minet.mokshitjain.co/api/format-message', {
  method: "POST",
  body: JSON.stringify({
    publicAddress: "",
    activities: [{ ... }],
  })
}).then(r => r.json())
```
